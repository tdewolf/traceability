<p>In the Data Governance Center, you can display an asset's traceability.</p><p>With this feature, you can trace:</p><ul style="list-style-type: square;"><li>Relationships between assets</li><li>People involved with a specific asset</li><li>References in other assets</li></ul><ac:structured-macro ac:name="info"><ac:rich-text-body><p>Not every relation that is displayed in an asset's overview tab is taken into account for the traceability.</p><ul><li>The Overview tab displays each relation type that is defined in the attribute assignment group of the asset type.</li><li>The Traceability tab displays each relation type that is defined in the traceability assignment group of the asset type.</li></ul></ac:rich-text-body></ac:structured-macro><p><span style="line-height: 1.4285715;"><br />Location: <strong>Asset page &gt; Traceability</strong></span></p>
